# Wabisabi SushiDerby 비공식 한글 번역 패치

## 프로젝트 구성

본 번역 패치는 치지직 스트리머 **여까**님의 커뮤니티에서 모집된 인원들로 구성된 번역팀 **글냥이**를 중심으로 제작되었습니다.

번역 및 패치 제작: 팀 글냥이 (Nark, Oatone)

팀 글냥이는 여까님과 커뮤니티 구성원들이 함께 참여하는 비공식 팬 번역팀입니다.

본 프로젝트는 개인 팬 활동의 일환으로 자발적으로 진행된 비공식 한국어 번역 작업입니다.

## 설치 방법

1. GitHub 페이지 우측의 **[Releases](../../releases)** 탭(혹은 이 링크)으로 이동합니다.
2. 최신 릴리즈에서 `Wabisabi-KRPatch.exe` 파일을 다운로드합니다.
3. 다운로드한 파일을 실행한 뒤, 게임 설치 경로가 자동으로 인식되지 않았다면 '자동 감지'나 '찾아보기' 버튼을 통해 게임 설치 경로를 설정해 주십시오.
4. 이후 우측 하단의 '한글 패치 설치' 버튼을 누르면 패치가 완료됩니다.
5. 게임 시작 후 **언어를 일본어로 설정한 뒤**, 텍스트가 한글로 출력되면 정상적으로 적용된 것입니다.

## 주의사항

- 이 패치는 일본어 파일을 덮어씌우기 때문에, 언어를 일본어로 설정해야 한국어가 표기됩니다.
- 원본 파일은 반드시 백업해두세요.
- 본 번역 패치 사용으로 인한 게임 오류나 문제는 책임지지 않습니다.

> - 본 저장소는 원본 게임의 내용을 재구성할 수 있는 형태의 데이터 또는 원본 아카이브의 추출물을 제공하지 않습니다.

> - 패치를 사용하려면 사용자가 별도로 정품 게임을 합법적으로 보유하고 있어야 합니다.
> - 원본 게임의 저작권 및 모든 권리는 해당 권리자에게 있습니다.
> - 본 번역물은 원작을 기반으로 한 파생 저작물에 해당하며, 원작의 이용 약관 및 관련 법률의 적용을 받을 수 있습니다.

> **리버스 엔지니어링 및 이용 범위**

> - 본 프로젝트는 게임의 복제 방지 장치 우회, 불법 복제, 또는 원본 게임의 무단 배포를 지원하거나 장려할 의도가 없습니다.
